## Hoe gebruik je deze repository?

1. Klik op **'Use this template'** → 'Create a new repository'
2. Kies je eigen GitHub-account als eigenaar
3. Geef de repository een naam: `react-lj2-[jouw-naam]`
4. Clone de repository naar je computer
5. Open de juiste opdrachtfolder in je editor
6. Voer `npm install` uit in de terminal
7. Start de app met `npm run dev`

## Inleveren

1. Commit & Push je werk naar GitHub
2. Kopieer de link naar je repository
3. Lever de link in via Canvas
