## Opdracht 09 – Flashcard Quiz

Maak een flashcard quiz over hoofdsteden. Op de voorkant van een kaartje staat een vraag, op de achterkant het antwoord. In totaal maak je 20 flashcards.

**Benodigdheden:**

- Een JS-bestand met 20 objecten, elk met een `question` en een `answer` property
- Component `FlashCard.jsx`:
  - Toont de voorkant of achterkant afhankelijk van een state
  - Gebruik conditionals om te wisselen bij klikken op de kaart
- Component `FlashCardList.jsx`:
  - Laadt alle `FlashCard`-componenten in
- `App.jsx`:
  - Laadt `FlashCardList` in en geeft de data door
- Styling met Tailwind CSS passend bij het thema van de quiz

**Theorie:**
- [React - Conditionals](https://meesterjson.nl/cheat-sheet/pages/react/conditionals.html)
- [React - Tailwind](https://meesterjson.nl/cheat-sheet/pages/react/tailwind.html)
