# Opdracht 9 – Tailwind

**Folder:** `opdracht-09-tailwind`

**Setup:**
```bash
cd opdracht-09-tailwind
npm install
npm run dev
```

---

**Opdracht 1:** Maak de pagina na die in Canvas staat. Gebruik uitsluitend Tailwind CSS, geen eigen CSS.

**Opdracht 2:** Voeg een dark theme toe. Als je laptop in dark mode staat, past de website mee.

> 📖 [Tailwind dark mode docs](https://tailwindcss.com/docs/dark-mode)

**Theorie:**
- [React - Tailwind](https://meesterjson.nl/cheat-sheet/pages/react/tailwind.html)

**Oplevering:** Commit & Push + een screenshot van de light en dark theme inleveren via Canvas
